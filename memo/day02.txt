HTML의 기본 요소
1. 태그(tag)
        <>로 표시한다.
        일반적으로 <시작></종료> 한 쌍으로 사용된다.
        일부 태그는 닫는태그가 존재하지 않는다. <태그이름/>

2. 내용(content)
        시작 태그와 종료 태그 사이에 있는 내용
        <p>내용</p>
3. 앨리먼트(element)
        태그  + 내용

4. 속성(attribute)과 속성값(value)
        태그 <br> 안에 있는 일종의 옵션(설정)
        속성 = "속성값"
        <h1 align="right">제목이다</h1>
        여러 속성을 지정 할 경우 각 속성을 띄어쓰기로 구분한다.
        <h1 align="right" id="test">제목이다</h1>

태그의 종류
        - <h1></h1> ~ <h6></h6> :  제목태그 , 숫자가 작을수록 글씨가 커진다
        - <p></p> : 단락태그, 단락은 위/아래 간격이 벌어지며 자동으로 줄이 바뀐다.
        - <strong></strong>, <b></b> : 굵은글씨
        - <br/> : 줄 바꿈 태그, 닫는 태그가 없다.
         - <hr/> : 가로선을 그려줌, 닫는 태그가 없다.
         * 닫는태그가 없는 것을 '빈 태그' 라고 부른다.

리스트 태그
        - <ol></ol>
            order list의 약자, 순서있는 리스트를 만듦
        - <ul></ul>
            Unorder List의 약자, 순서 없는 리스트를 만듦
        - <li></li>
            List의 약자, ol 또는 ul내부에서 사용


테이블 태그(표 만들기)

        <table></table> : 표를 만드는 태그
                - 테이블 태그는 표를 만들 때 사용하며 표의 제목, 행의 수 , 열의 수 등을 다른 태그를 이용해 표현한다.

                -table 태그 내부에서 사용 가능한 태그는 다음과 같다.
                <caption></caption> : 해당 표의 제목
                
                <tr></tr> : 행을 만드는 태그 
                        colspanm rowspan 속성을 사용한다.
                
                <th></th>

이미지 삽입 태그
        <img src="이미지 파일의 경로" alt="이미지 설명">
         src 속성은 넣고 싶은 이미지의 경로를 입력한다.
         웹 상의 주소를 입력해도 된다.

         링크를 거는 a태그(앵커 태그)
                <a href="링크나 경로">링크를 걸어 줄 컨텐츠</a>
                a태그로 감싼 요소는 해당 경로로 링크가 걸린다.

인라인 / 블록 요소
        <sapn></sapn> : 대표적인 인라인 태그 
        <div></div> : 대표적인 블록 태그

        - sapn과 div태그는 화면의 레이아웃을 나누는데 주로 사용되며 중요하다!
        - 둘 다 의미가 없다.(시멘틱하지 않은 태그)

요소를 여러 관점으로 분류할 수 있다.
그 중 중요하게 봐야할 것은 인라인과 블록이다.

인라인 요소
        - 내용물(컨텐츠)의 크기만큼만 영역을 차지하므로 다른 요소와 같은줄에 위치할 수 있다.
          즉, 줄 바꿈이 되지 않는다.

블록 요소
        - 내용물(컨텐츠)과 상관없이 한 줄을 다 차지하므로 다른 요소와 같은 줄에 위치할 수 없다.
          즉, 너비 속성이 100%를 다 차지하고 자동으로 줄 바꿈이 된다.

css(Cascading Style Sheets)

HTML로 만든 웹 문서의 스타일(디자인)을 담당한다.
글꼴, 색상 테두리, 정렬, 배치(레이아웃) 등의 외관을 담당

css형식
        속성 : 속성값1;
        
        속성은 미리 정해져 있으며 필요에 따라 검색하여 사용한다.

스타일 적용하는  방법은 크게 3가지가 있다.
        1. 인라인 스타일
                스타일을 적용시킬 태그에 style 속성을 사용한다.
                <p style = "color : red; background-color : blue;">내용</p>

                인라인 스타일은 간단하게 사용하기에는 좋으나 코드가 길어지 수록 가독성이 떨어지며
                css와 html이 뒤섞여 유지보수가 매우 힘들어진다.
        
        2. 내부 스타일
                <head></head> 영역에 <style></style> 태그를 사용하여 각 요소들의 스타일을 정의한다,.
                형식은 다음과 같다.

                선택자{
                        속성1 : 속성값1;
                        속성2 : 속성값2;
                }

                선택자 :  어떤 요소를 선택하기 위해 사용하는 것.

        3. 외부 스타일
                별도의 css파일을 만들어서 적용시킨다.
                css와 html을 완전히 분리하여 관리할 수 있기 때문에 가장 많이 사용하는 방식
                 css파일을 html파일에 적용시키기 위해서는 <head></head>영역에 
                 <link>태그를 사용하면  된다

                 <link rel = "stylesheet" href = "파일경로">


        id와 class
                id와 class 속성은 특정 태그들을 식별하고 그룹화 하는데 사용한다.
                모든 태그에 사용 가능한 속성이다.
                (쉽게 말하면 태그에 이름을 붙이는 것)

                - id
                  특정 태그에 식별 가능한 고유값을 부여한다,
                  선택자로 사용할 때는 #을 붙인다.

                - class
                  여러 태그를 하나의 집단으로 묶어줌
                  선택자로 사용할 때는 .을 붙인다.

        하나의 태그에 id와 class를 동시에 가질 수 있으며 class를 여러개 가질 수 도  있다.
        주로 class를 사용하고 id는 그 페이지에소 중요하고 유일한 요소에게 부여한다.
